# kineticorganization.com

Canonical home of the doctrine.
- No funnels, no popups, no social proof.
- Designed for skeptical senior readers.
- Explicit version discipline (footer + Chapter 00 page note).

## Stack (default)
- Eleventy (11ty) static site generator
- GitHub Pages (via Actions)

## Local dev
```bash
cd site
npm install
npm run dev
```

## Versioning
- `VERSION` is the single source of truth (e.g. v2.1).
- Footer renders version string.
- Chapter 00 page includes an explicit entry-point note.
